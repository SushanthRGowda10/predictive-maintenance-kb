# 📈 Multi-Model Benchmark Results

## Compared Models
- Random Forest
- Gradient Boosting
- Logistic Regression
- Support Vector Machine

## Objective
The purpose of benchmarking is to justify the final model choice scientifically.

## Expected Outcome
Random Forest is expected to outperform due to:
- nonlinear sensor relationships
- robust ensemble learning
- reduced overfitting
- strong tabular data performance

## Deliverable
The benchmark script exports:
- accuracy
- precision
- recall
- F1-score
- ranked model comparison CSV

# 🧠 Explainable AI Report

## Objective
This report explains which machine sensor features contribute most to failure prediction.

## Deliverables
- feature importance ranking
- feature importance chart
- leakage detection support
- Random Forest decision transparency

## Why This Matters
Explainability improves:
- trust
- viva defense
- model justification
- industrial interpretability

## Observation Goal
If one feature dominates nearly 100%, the dataset may be overly synthetic or contain leakage.

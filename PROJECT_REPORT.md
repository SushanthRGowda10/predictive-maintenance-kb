# 📘 Predictive Maintenance Knowledge Base — Final Report

## Abstract
This project presents an AI-driven predictive maintenance system that predicts machine failures using Random Forest and provides knowledge-based maintenance recommendations through a Streamlit BI dashboard.

## Introduction
Traditional maintenance approaches are reactive and expensive. This project introduces proactive failure prediction using machine learning.

## Problem Statement
Unexpected industrial machine failures increase downtime and maintenance costs.

## Objectives
- Predict failures early
- Reduce downtime
- Optimize maintenance cost
- Improve fleet visibility

## Dataset
Industrial machine sensor dataset containing:
- temperature
- vibration
- pressure
- runtime_hours
- error_count
- maintenance_history

## Methodology
Dataset → preprocessing → model training → prediction → knowledge base → dashboard → alerts

## Results
- 96.4% accuracy
- fleet-level risk visibility
- maintenance recommendation engine
- downtime estimation

## Future Scope
- IoT sensors
- cloud deployment
- anomaly detection
- LSTM forecasting

## Conclusion
The system successfully combines ML + BI + knowledge recommendations for smart manufacturing.

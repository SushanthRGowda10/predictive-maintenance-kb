# 🔧 Predictive Maintenance Knowledge Base

An **AI-driven predictive maintenance dashboard** that predicts machine failure risk, estimates downtime, and provides maintenance recommendations using **Machine Learning + Business Intelligence dashboards**.

---

## 🚀 Features
- ✅ Machine failure prediction using Random Forest
- ✅ Severity-based maintenance recommendations
- ✅ Fleet monitoring dashboard
- ✅ Maintenance cost estimation
- ✅ Downtime prediction
- ✅ Critical machine alerts
- ✅ Risk distribution histogram
- ✅ Temperature trend chart
- ✅ KPI cards for operations monitoring

---

## 🧠 Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Joblib

# 📈 Sensor Forecasting Report

## Objective
Forecast future machine temperature trends for the next 24 runtime hours.

## Method
A linear regression forecasting model is used on:
- runtime_hours
- temperature

## Deliverables
- temperature_forecast.csv
- temperature_forecast.png

## Engineering Value
This layer upgrades the project from:
- current failure prediction
to
- future condition intelligence

This supports proactive maintenance scheduling.
